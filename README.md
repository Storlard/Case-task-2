# Case-task-2

Простое веб-приложение-счетчик на чистом JavaScript.

## Описание
Counter App позволяет увеличивать и уменьшать значение с помощью кнопок.

Приложение демонстрирует базовую работу с:
- DOM
- слушателями событий
- условной логикой
- динамическим изменением стилей

---
## Функционал

- Увеличение и уменьшения значения
- Ограничение диапозона: от -10 до +10
- Блокировка кнопок при достижении лимита
- Изменение цвета в зависимости от значения:
  - 0 - красный
  - > 0 - желтый
  - < 0 - зеленый
- Сообщение при достижении крайних значений
